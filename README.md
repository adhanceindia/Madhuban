# Madhuban Garden Resort Website

Frontend-first Next.js 14 project for **Madhuban Garden Resort** in **Agar Malwa District, Madhya Pradesh, India**.

## Current Phase

- Phase 1: frontend with dummy data only
- No database connections
- No API routes
- No payment integration yet

## Stack

- Next.js 14 App Router
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Lucide React
- ESLint + Prettier

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Project Notes

- Global project instructions live in [AGENTS.md](./AGENTS.md)
